README.md - Vending Machine MK1
Short Functionality of the Code
This project is a vending machine simulation built using Python and Tkinter. Users can browse items by category, view product conditions (such as temperature), and purchase items. The system manages inventory and records sales using a SQLite database.
Software Required to Run the Code
Python 3.x
Tkinter (comes pre-installed with Python)
SQLite3 (included with Python)
IDE (recommended: PyCharm)
Required Changes Before Running
Make sure the database file is correctly set up.
If needed, update the database path inside db.py.
Ensure all image assets are in the correct folder:
- snow.png
- sun.png
- leaf.png
- warning.png
- redx.png
How to Run the Code
1. Clone the repository:
https://github.com/David8836/Vending-Machine-MK1

2. Open the project in your IDE

3. Run the main file:
python main.py
