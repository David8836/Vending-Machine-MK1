Vending Machine MK1

Project Description
This project is a smart vending machine system that connects through an integrated network.

The vending machine has three sections:
- Snacks
- Health
- Electronics

Features
- Tracks inventory levels
- Tracks cash levels
- Supports multi-item purchases
- Sends alerts when stock is low
- Sends machine status updates
- Central management dashboard support

Technologies Used
- Python
- Networking concepts
- Error handling
- Real-time data synchronization

Target Audience
- Small businesses
- Campus facilities
- Offices
- Hospitals
- Public spaces
